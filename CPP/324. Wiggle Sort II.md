# [324. Wiggle Sort II](https://leetcode.com/problems/wiggle-sort-ii/description/)
* 参考 http://www.cnblogs.com/grandyang/p/5139057.html

```c++
    void wiggleSort(vector<int>& nums) {
        vector<int> tmp = nums;
        int n = nums.size();
        int k = (n+1)/2;
        int j = n;
        sort(tmp.begin(), tmp.end());
        for(int i=0; i<n; i++){
            nums[i] = i&1 ? tmp[--j] : tmp[--k];
        }
    }
```
