# [283. Move Zeroes](https://leetcode.com/problems/move-zeroes/?tab=Solutions)
* 第一反应的解法：类似冒泡排序

```C++
// 用冒泡排序法
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int n = nums.size();
        n--;
        int j = 0;
        while(n > 0){
            j = 0;
            for(int i = 0; i< n; i++){
                if(nums[i] == 0){
                    swap(nums[i], nums[i+1]);
                    j = i;
                }
            }
            
            n = j;
        }
    }
};
```

* O(n)的解法

```C++
// 把非零元素都往前copy，记录下最后非零元素的对应的位置，后面的全部变为0
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int n = nums.size();
        int j = 0;
        for(int i = 0; i< n; i++){
            if(nums[i] != 0){
                nums[j++] = nums[i];
            }
        }
        
        for(; j < n; j++){
            nums[j] = 0;
        }
    }
};
```
