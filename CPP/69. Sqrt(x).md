# [69. Sqrt(x)](https://leetcode.com/problems/sqrtx/#/solutions)
* 就是利用二分查找

```python
class Solution(object):
    def mySqrt(self, x):
        """
        :type x: int
        :rtype: int
        """
        if x == 0: return 0
        left , right =1, x
        
        while True:
            mid = (left + right) /2
            if mid * mid > x:
                right = mid-1
            else:
                if (mid+1) * (mid+1) > x:
                    return mid
                left = mid+1
								
```

```c++
class Solution {
public:
    int mySqrt(int x) {
        if(x <= 1) return x;
        int left =1, right = INT_MAX;
        while(true){
            int mid = left + (right-left)/2;
            if(mid > x / mid){
                right = mid-1;
            }else{
                if(mid + 1 > x/(mid+1)){
                     return mid;
                }     
                left = mid+1;
            }
        }
    }
};
```
