* 标准二分查找: 1. 判断条件是 l<=h   2. 每次都是更新l和h，并且是 +1 和 -1

```java
int binarySearch(int[] nums, int target){
      l = 0, r = nums.length-1;
      while(l <= h){
          int mid = l  + (h - l) /2;
          if(nums[mid] == target){
             return mid;
          }
          
          if(nums[mid] < target){
              l = mid + 1;
          }else {
              h = mid -1;
          }
          
          return -1;
      }
}
```
